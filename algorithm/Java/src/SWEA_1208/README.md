Flatten

소요시간 50분

SWEA 첫 자바 풀이였다.
input 스트림 연결, class명, static 변수 사용 등을 익혔다.

문제 풀이 자체는 어렵지 않았다.

그런데 더 좋은 풀이가 있었다.

나는 박스들의 높이를 정렬한 후 높은 박스를 하나씩 내려주는 방식으로 풀이했고,

가장 짧은 실행시간의 코드를 보니, 박스의 높이를 index로 하는 배열을 썼다.

왜 이걸 생각 못했나 했고, 앞으론 변수의 범위를 잘 보고, 배열의 index로 쓸 수 있는지 확인해야겠다.
